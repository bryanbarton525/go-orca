# Blog Post Style Guide

## Word Count Targets

| Post type       | Target range  |
|-----------------|---------------|
| Quick tip       | 400–600 words |
| Tutorial        | 800–1 500 words |
| Deep dive       | 1 500–3 000 words |
| Retrospective   | 600–1 200 words |

## Opening Paragraph Rules

- Do not start with "In this post, we will…"
- Lead with the result or the pain point, not the agenda.
- Three sentences max for the hook.

## Common Mistakes to Avoid

1. **Passive-voice conclusions**: "It was shown that X is faster" → "X is 3× faster."
2. **Orphaned mermaid diagrams**: every diagram needs a caption explaining what to look for.
3. **Missing transition sentences**: each section should have a one-sentence bridge to the next.
4. **Over-qualifying**: "It might potentially be possible that…" → delete hedges unless accuracy requires them.
5. **Code without context**: show the problem code before the solution code.
